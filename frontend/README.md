# LearnMate Frontend

LearnMate前端应用，基于React + Vite + Ant Design构建。

## 功能特性

- 📄 文档上传和管理
- 🔍 文档列表查看
- 💬 AI智能问答界面
- 🎨 现代化的UI设计

## 安装依赖

```bash
npm install
# 或
yarn install
# 或
pnpm install
```

## 开发

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
```

应用将在 http://localhost:3000 启动

## 构建

```bash
npm run build
# 或
yarn build
# 或
pnpm build
```

## 环境变量

创建 `.env` 文件（可选）：

```env
VITE_API_BASE_URL=http://localhost:8000
```

## 技术栈

- React 18
- Vite
- Ant Design 5
- React Router
- Axios
